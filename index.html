Privacy Policy

Last updated: March 11, 2026

Profile Lock ("the Extension") is a Chrome browser extension that allows users to password-protect their Chrome profiles. This privacy policy explains how we handle your data.

In short: Profile Lock does not collect, transmit, or share any personal data. All information stays on your device.

1. Data Collection

Profile Lock does not collect any personal information. We do not collect:

  • Names, email addresses, or any personally identifiable information
  • Browsing history or web activity
  • IP addresses or location data
  • Analytics, telemetry, or usage statistics

2. Data Storage

The only data stored by the Extension is:

  • Password hash — Your password is converted into a SHA-256 hash (a one-way cryptographic digest) before being stored. The original password is never saved.
  • Lock state — A boolean flag indicating whether the profile is currently locked.

All data is stored locally on your device using Chrome's chrome.storage.local API. This data is specific to each Chrome profile and is never transmitted to any external server.

3. Data Sharing

Profile Lock does not share any data with third parties. There are no external servers, APIs, analytics services, or advertising networks involved.

4. Permissions

The Extension requires the following Chrome permissions:

  • storage — To save the password hash and lock state locally on your device.
  • tabs — To display the lock screen and manage tab behavior when the profile is locked.
  • alarms — Reserved for potential future features such as auto-lock timers.

5. Security

We take reasonable measures to protect your data:

  • Passwords are stored as SHA-256 hashes, not in plain text.
  • All data remains local to your device and Chrome profile.
  • No network requests are made by the Extension.

6. Children's Privacy

Profile Lock does not knowingly collect data from children under 13. Since no personal data is collected at all, the Extension is safe for use by all ages.

7. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date at the top of this page.
